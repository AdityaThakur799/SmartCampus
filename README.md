# SmartCampus - Student Management System

## Overview

SmartCampus is a Django-based Student Management System developed as an MCA portfolio project. The system helps educational institutions manage students, staff, courses, attendance, and results through a centralized web application.

## Features

### Admin Module

* Manage Students
* Manage Staff
* Manage Courses
* Manage Subjects
* Manage Attendance
* Manage Results
* View Reports

### Staff Module

* Take Attendance
* Update Student Results
* Manage Student Records
* View Attendance Reports

### Student Module

* View Attendance
* View Results
* Update Profile
* Apply for Leave

## Installation

```bash
git clone https://github.com/AdityaThakur799/SmartCampus.git
cd SmartCampus
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
